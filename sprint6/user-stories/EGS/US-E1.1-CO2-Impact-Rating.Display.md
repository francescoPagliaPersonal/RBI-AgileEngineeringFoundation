# US E1.1: CO₂ Impact Rating Display
As an environmentally conscious craftsman
I want to see a simple CO₂ rating (A-E scale) for each tool
So that I can identify more sustainable options for my business

## Acceptance Criteria:
- CO₂ rating (A = low impact, E = high impact) displays on product cards
- Rating based on simple category logic (recycled materials = B, steel = D)
- Hover tooltip explains: "A = most environmentally friendly, E = higher environmental impact"
- Feature can be toggled on/off via admin panel (for A/B testing)
- Rating displays correctly in mobile view
- Analytics track user interactions with ratings
